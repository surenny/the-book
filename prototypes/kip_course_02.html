过滤、关联分次、动机问题

上一课结尾的那道难题——"双层链复形的同调能不能从层的同调拼出来"——其实是谱序列存在的根本理由。这一课我们把这个困惑彻底点破，再给"谱序列要解决的问题"一个干净的提法。

本课目标。 读完之后你应当能：(1) 给出一个具体反例，看到 $H_*(\mathrm{gr}^\bullet C) \neq H_*(C)$ 是常态而非例外；(2) 写出"过滤的链复形"和"关联分次"的精确定义；(3) 把"我们究竟想算什么"用一句话说清，并且能解释 gr 丢了哪两类信息。

2.1从上一课的引子说起

上次课最后一题让你想：

设 $C_n = F_n \oplus G_n$，$d_n$ 把 $G$ 的元素送到 $G$（即 $G$ 是子复形），但可能把 $F$ 的元素送到 $F \oplus G$（"混合"到 $G$ 里）。$H_*(C)$ 与 $H_*(F)$、$H_*(G)$ 是什么关系？

"觉得不像练习 2 那么简单"是对的。下面这个具体例子让你看清楚为什么——它就是经典代数拓扑里的 恒等映射的锥（mapping cone of identity）。

关键反例 · $\mathrm{Cone}(\mathrm{id}_\mathbb{Z})$

设 $C$ 是这样一个链复形：

$$\cdots \to 0 \to \underbrace{\mathbb{Z}}_{C_1} \xrightarrow{\;\mathrm{id}\;} \underbrace{\mathbb{Z}}_{C_0} \to 0 \to \cdots$$

令 $G$ 是 $C$ 在底层的子复形：$G_0 = \mathbb{Z},\ G_1 = 0$（其他都是零）。这是合法的子复形——$d$ 限制在 $G$ 上是零映射，自然落在 $G$ 里。

商 $F := C/G$ 在底层为零、在 $C_1$ 处是 $\mathbb{Z}$、$d = 0$。

所以拆分 $C_n = F_n \oplus G_n$ 成立，且符合上题的 setup（$G$ 是子复形，$d$ "混合" $F$ 到 $G$，因为 $d_1$ 把 $C_1 = F_1$ 中的生成元送到 $C_0 = G_0$）。

分别算同调：

  • $H_*(F)$：$d = 0$ 在 $F$ 上，所以 $H_1(F) = \mathbb{Z},\ H_0(F) = 0$。
  • $H_*(G)$：$d = 0$ 在 $G$ 上，所以 $H_0(G) = \mathbb{Z},\ H_1(G) = 0$。
  • "拼起来"：$H_1 \oplus = \mathbb{Z},\ H_0 \oplus = \mathbb{Z}$。总秩 2。

算原复形的同调：

  • $\ker d_1 = 0$（恒等映射是单的），所以 $H_1(C) = 0$。
  • $\ker d_0 = \mathbb{Z}$，$\mathrm{im}\, d_1 = \mathbb{Z}$（满），所以 $H_0(C) = 0$。
  • $H_*(C) = 0$。总秩 0。

差了整整 2 个 $\mathbb{Z}$——这个差不是小修小补，是定性的差异。

所以朴素的"把每层同调拼起来"是错的。层间的混合（mixing）信息在拆开看每层时被完全丢失了。这正是谱序列要弥补的事。

2.2把"分层"的概念抽象出来：过滤

上例中的"两层结构"其实是一种特殊情形。一般地我们这样定义：

定义 · 链复形上的过滤

设 $(C_\bullet, d)$ 是一个链复形。一个 降序过滤（descending filtration） 是一族子复形

$$\cdots \;\supseteq\; F^{p-1} C \;\supseteq\; F^p C \;\supseteq\; F^{p+1} C \;\supseteq\; \cdots$$

满足每个 $F^p C$ 是子复形——即 $d(F^p_n) \subseteq F^p_{n-1}$，微分保持过滤。

通常假设 有界（bounded）：存在 $p_{\min}$ 使 $F^{p_{\min}} C = C$，存在 $p_{\max}$ 使 $F^{p_{\max}+1} C = 0$。这一节我们都默认有界。

有了过滤，我们就有"$p$ 越大代表越深、越小"的尺度。把它想成"层级"——$F^0$ 是整个复形，$F^1$ 是它里面更深的一层，$F^2$ 是更深的一层，…，最深的是零。

几个你一定要见过的例子：

例 a · 平凡过滤

$F^0 C = C$，$F^1 C = 0$。一切都在第 0 层。这种过滤的谱序列就是"没什么发生"——直接 $H_*(\mathrm{gr}) = H_*(C)$。

例 b · 两步过滤（上面的反例就是这一类）

$F^0 C = C \supseteq F^1 C = G \supseteq F^2 C = 0$，其中 $G$ 是 $C$ 的子复形。

例 c · 双复形（bicomplex）的列过滤

设 $C^{\bullet, \bullet}$ 是一个双复形——每个 $C^{p, q}$ 上有水平微分 $d_h: C^{p,q} \to C^{p+1, q}$ 和垂直微分 $d_v: C^{p,q} \to C^{p, q-1}$（取 homological 约定使 $d_v$ 降阶），且 $d_h \circ d_v + d_v \circ d_h = 0$（反对易）。

定义总复形 $\mathrm{Tot}^n = \bigoplus_{p+q=n} C^{p,q}$，总微分 $d = d_h + (-1)^p d_v$ 满足 $d^2 = 0$。

"列过滤"取 $F^p \mathrm{Tot} = \bigoplus_{p' \geq p,\, q} C^{p', q}$——只留下水平指标 $\geq p$ 的部分。每层是子复形（$d_h, d_v$ 都不会降低 $p$）。

双复形是产生谱序列最常见的来源。下次课的例子就是它。

例 d · CW 复形的骨架过滤

拓扑空间 $X$ 如果有 CW 结构，它的 $n$-骨架（$n$-skeleton） $X^{(n)}$ 是一族升序子空间 $X^{(0)} \subseteq X^{(1)} \subseteq \cdots$。把这翻译到链复形上就得到一个升序过滤（也可以反转成降序）。这就是 Serre 谱序列、Atiyah–Hirzebruch 谱序列等等的来源。

2.3关联分次：把"层间混合"忘掉

定义 · 关联分次复形

设 $C$ 带过滤 $F^\bullet$。它的 关联分次（associated graded）是分次链复形

$$\mathrm{gr}^p C \;:=\; F^p C \,/\, F^{p+1} C, \qquad \mathrm{gr}^\bullet C \;:=\; \bigoplus_{p} \mathrm{gr}^p C.$$

每个 $\mathrm{gr}^p C$ 仍然是一个链复形——因为 $d$ 保持过滤，所以 $d$ 诱导出商上的差分；$d^2 = 0$ 也自动传递。

直观上，$\mathrm{gr}^\bullet C$ 是 $C$ 的"层与层之间的连接被忘掉"的版本。它在某种意义上"更简单"——每层独立，互不干扰。

回到反例

把 §2.1 的 $C = \mathrm{Cone}(\mathrm{id}_\mathbb{Z})$ 用 $F^0 = C \supseteq F^1 = G \supseteq F^2 = 0$ 过滤：

  • $\mathrm{gr}^0 = C/G$：在 deg 1 处是 $\mathbb{Z}$；微分 $= 0$（因为 $d_1$ 把元素送到 $G$，落到 $\mathrm{gr}^0$ 时就是 0 了）。
  • $\mathrm{gr}^1 = G$：在 deg 0 处是 $\mathbb{Z}$；微分 $= 0$。

$\mathrm{gr}^\bullet C$ 总共有两个非零的 $\mathbb{Z}$，分别在 (filtration $0$, degree $1$) 和 (filtration $1$, degree $0$)。两者都自动是 cycle（因为各自微分都为零）、都不是 boundary。所以 $H_*(\mathrm{gr}^\bullet C)$ 有两个 $\mathbb{Z}$。

而 $H_*(C) = 0$。差距正好两个 $\mathbb{Z}$。

2.4中心问题

现在我们可以把整门课要回答的问题写出来：

核心问题。 给定一个过滤链复形 $(C, d, F^\bullet)$，仅凭关联分次 $\mathrm{gr}^\bullet C$ 的信息（这通常比 $C$ 本身简单很多），能在多大程度上重建 $H_*(C)$？

从反例可知，不能直接重建：$H_*(\mathrm{gr}^\bullet C) \neq H_*(C)$。但 $H_*(\mathrm{gr}^\bullet C)$ 仍然是 $H_*(C)$ 的第一近似——剩下的事情是把它一步步修正。

谱序列就是这个"逐步修正"的引擎。下面我们看清"gr 究竟丢了什么信息"，这会让下一课的构造水到渠成。

2.5gr 丢了哪两类信息

回到反例，仔细想想 $\mathrm{gr}^\bullet C$ 的两个"伪"同调类来源：

失败模式 ①：gr-cycle 但不是真 cycle

看 $\mathrm{gr}^0 = (C/G)$ 在 deg 1 的生成元 $\bar{1}$。它在 $\mathrm{gr}^0$ 里是 cycle，因为差分（在 $\mathrm{gr}^0$ 中）为零。

但回到 $C$ 里，它对应一个 $1 \in C_1$，而 $d_1(1) = 1 \neq 0$。真 $C$ 里它根本不是 cycle，只是它的 $d$ 落到了下一层 $G$ 里，被 $\mathrm{gr}^0$ 视作"不可见"，于是看上去是 cycle。

抽象地说：$x \in F^p C$ 是 gr-cycle $\iff dx \in F^{p+1} C$（下沉一级）。但要它是真 cycle，需要 $dx = 0$（一路下沉到底）。"中间状态" $dx \in F^{p+k} C$ 但 $\neq 0$，gr 看不见。

失败模式 ②：gr-cycle 是真 cycle，但是个真 boundary，gr 看不见它被边界过

看 $\mathrm{gr}^1 = G$ 在 deg 0 的生成元 $1 \in G_0$。在 $\mathrm{gr}^1$ 里它是 cycle（差分为零），而且在 $C$ 里也是真 cycle（$d_0 = 0$）。

但是！在 $C$ 里它是一个 boundary——$1 = d_1(1)$，其中 $d_1$ 的源 $1 \in C_1$ 来自上一层 $F^0 \setminus F^1$。$\mathrm{gr}^1$ 看不到 $F^0$ 里的东西，所以无法发现它被边界过。

抽象地说：$x \in F^p C$ 是 boundary $\iff x = dy$ for some $y \in C$。如果 $y \in F^{p-1} C \setminus F^p C$（来自上一层），$\mathrm{gr}^p$ 看不到 $y$，于是错过 $x$ 作为 boundary 的身份。

归纳起来。 gr 看东西的窗口太窄：cycle 条件被放宽了（"下沉一级"代替"等于零"），boundary 条件被收紧了（只看"同层内的边界"）。两者都让 gr 比 $C$ 多出"假的同调类"。

2.6下一课要做的事

上面两个失败模式给了我们一个非常清晰的修正思路。我们要做两件事：

  1. 把 cycle 的概念分级。
    • "$1$-级 cycle"：$x \in F^p C$ 且 $dx \in F^{p+1} C$（即 gr-cycle）；
    • "$2$-级 cycle"：$x$ 且 $dx \in F^{p+2} C$（更深）；
    • "$r$-级 cycle"：$x$ 且 $dx \in F^{p+r} C$（更更深）；
    • "$\infty$-级 cycle"：$x$ 且 $dx = 0$（真 cycle）。
  2. 把 boundary 的概念也分级。
    • "$1$-级 boundary"：$x \in F^p C$ 是 $F^p$ 内的某个东西的边界；
    • "$2$-级 boundary"：$x \in F^p C$ 是 $F^{p-1}$ 里某个东西的边界；
    • "$r$-级 boundary"：是 $F^{p-r+1}$ 里某个东西的边界；
    • "$\infty$-级 boundary"：是任何 $C$ 里东西的边界（真 boundary）。

取 cycle / boundary 的商，就给出 $r$-级近似——这就是 谱序列的第 $r$ 页 $E_r$：

$$E_r^{p,\bullet} \;=\; \frac{\{r\text{-级 cycle in } F^p\}}{\{r\text{-级 boundary in } F^p\}}.$$

$r = 1$ 时这就是 $H_*(\mathrm{gr}^p C)$，即 $E_1 = H_*(\mathrm{gr}^\bullet)$。$r = \infty$ 时正好给出 $H_*(C)$ 的关联分次（这就是 收敛，第 4 课的话题）。

而且，惊人的是，从第 $r$ 页 能用一个自然定义的微分 $d_r$ 得到第 $r+1$ 页，整个过程像一台精炼机器，一级级把"假同调类"洗掉。

第 3 课我们就来精确造出这个机器。

2.7这一课的小结

  1. 过滤是链复形上的一族递降子复形 $F^\bullet C$。
  2. 关联分次 $\mathrm{gr}^p C = F^p / F^{p+1}$ 是把"层间混合"忘掉之后的复形。
  3. 一般 $H_*(\mathrm{gr}^\bullet C) \neq H_*(C)$。Cone-of-identity 反例展示了二者可以完全不同。
  4. gr 丢了两类信息：(a) 把 cycle 条件放宽了（dx 只需"下沉一级"），(b) 把 boundary 条件收紧了（看不到上一层来的边界）。
  5. 谱序列就是把 cycle / boundary 概念逐级精炼所得到的一族越来越精细的近似 $E_1 \to E_2 \to \cdots \to E_\infty$，最终（在收敛的情形下）给出 $H_*(C)$ 的关联分次。

2.8作业

练习 1 · 验算反例

用上面给出的过滤 $F^0 = C \supseteq F^1 = G \supseteq F^2 = 0$，把 $\mathrm{Cone}(\mathrm{id}_\mathbb{Z})$ 的所有 $H_*(\mathrm{gr}^p)$ 算一遍，确认拼起来共 2 个 $\mathbb{Z}$，而 $H_*(C) = 0$。然后用自己的话总结：那两个"假同调类"分别对应失败模式 ① 还是 ②？

练习 2 · 三步过滤

取 $C$ = ($\mathbb{Z}/4 \xrightarrow{\cdot 2} \mathbb{Z}/4$)（这是练习题第 1 课的对象）。设过滤为

$$F^0 = C, \quad F^1 = (\text{2-torsion 子复形}) = (\;0 \to 2\mathbb{Z}/4\;), \quad F^2 = 0.$$

验证 $F^1$ 是子复形，写出 $\mathrm{gr}^0,\ \mathrm{gr}^1$ 并分别算它们的同调；最后对比 $H_*(C)$（你应当已经算过）。

练习 3 · 引子题：$d_1$ 的雏形

设 $C$ 带过滤 $F^\bullet$，$x \in F^p C$ 是一个 gr-cycle——即 $dx \in F^{p+1} C$。注意：$dx$ 本身在 $C$ 里是非零的可能元素，但它落在 $F^{p+1}$ 里。

问题：$dx$ 在 $\mathrm{gr}^{p+1} C = F^{p+1}/F^{p+2}$ 中代表的那个类——叫做 $[dx] \in \mathrm{gr}^{p+1}_{n-1}$——是不是 $\mathrm{gr}^{p+1}$ 里的一个 cycle？为什么？

（这就是下一课谱序列的第一个微分 $d_1: E_1^p \to E_1^{p+1}$ 的形态。等你下次课见到 $d_1$ 的精确定义时，会发现就是这件事在做。）
