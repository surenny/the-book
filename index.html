The Book

"You don't have to believe in God, but you should believe in The Book." — Paul Erdős

Erdős imagined a book in which God keeps the most elegant proof of every theorem. This is its humbler cousin — formalization case studies, evaluation postmortems, system tech-reports, dashboards from AI4M work, rendered as interactive HTML so 200-line markdown stops being where ideas go to die.

Add a new report

每个 report 是一个独立 HTML 文件。两条公开度路径：

// 公开 — committed to git, deployed to GitHub Pages
//   file:     prototypes/your_report.html
//   registry: reports.json

// 本地 — gitignored, never leaves this machine
//   file:     private/your_report.html
//   registry: private/reports.local.json

{
  "id": "your_report_id",
  "path": "prototypes/your_report.html",  // or "private/..."
  "title": "…",
  "project": "FormalQualBench",
  "date": "2026-05-13",
  "type": "Case Study | Postmortem | Tech Report | Dashboard | Essay",
  "type_icon": "◆",
  "tags": ["…"],
  "deck": "一两句话总结",
  "highlights": ["3-4 个关键数字"]
}
