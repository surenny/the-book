The Book

"You don't have to believe in God, but you should believe in The Book." — Paul Erdős

Erdős imagined a book in which God keeps the most elegant proof of every theorem. This is its humbler cousin — formalization case studies, evaluation postmortems, system tech-reports, dashboards from AI4M work, rendered as interactive HTML so 200-line markdown stops being where ideas go to die.

Add a new report

每个 report 是一个独立 HTML 文件。放进 prototypes/，然后在 reports.json 的 reports 数组里追加一条：

{
  "id": "your_report_id",
  "path": "prototypes/your_report.html",
  "title": "…",
  "project": "FormalQualBench",
  "date": "2026-05-13",
  "type": "Case Study | Postmortem | Tech Report | Dashboard | …",
  "type_icon": "◆",
  "tags": ["…"],
  "deck": "一两句话总结",
  "highlights": ["3-4 个关键数字"]
}
